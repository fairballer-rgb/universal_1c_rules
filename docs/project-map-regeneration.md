# Регенерация `project-map` из комплекта

Используйте этот сценарий, если в целевом репозитории еще нет каталога `project-map` или карта устарела.

## Предусловия

- В репозитории есть XML-выгрузка конфигурации (минимум `Configuration.xml`, `Subsystems/`, `CommonModules/`, `Documents/`).
- Python 3.9+ доступен в терминале.
- Каталог `project-map/` скопирован в корень проекта.

## Быстрый запуск (из корня проекта)

```bash
python3 project-map/generate-map.py .
```

## Что должно появиться

- `project-map/PROJECT_MAP.md`
- `project-map/subsystems/*.md`

## Повторная регенерация

После изменения структуры конфигурации запустите ту же команду еще раз:

```bash
python3 project-map/generate-map.py .
```

Скрипт перезапишет индекс и детальные файлы подсистем актуальными данными.
